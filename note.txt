2). 对从页面传入的请求参数, 若要求该请求参数为整型的字符串, 类似于 "1", 在 Controller 中的建议处理方案:

	int id = -1;
	
	try {
		id = Integer.parseInt(idStr);
	} catch (NumberFormatException e) {}
	
	if(id > 0){
		book = bookService.getBook(id);
	}
